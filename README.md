# Apollo.io (apollo-io)

Apollo.io is a sales intelligence and engagement platform combining a 270M+ contact database with email and call sequencing. The Apollo REST API exposes people search & enrichment, organization search & enrichment, contacts, accounts, deals, sequences, email accounts, calls, tasks, meetings, lists, and webhooks.

## Plans

- **Free:** limited monthly credits, no API
- **Basic:** ~$59/seat/mo annual — REST API access starts here
- **Professional:** ~$99/seat/mo annual
- **Organization:** ~$149/seat/mo annual
- **Credit add-ons:** mobile, exports, dialer minutes, email sends

## Rate Limits

- **Per minute / per hour / per day** request caps scaled by plan
- **Search credits** + **enrichment credits** + **mobile credits** consumed from separate monthly pools
- **Bulk endpoints** (up to 10 records) cost less per record
- 429 returned when rate or credit pool exhausted
